# Personal Portfolio Starter (UX)

A lightweight single-page portfolio scaffold tailored for UX designers.

## Quick start

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.

## Customize content

Update `data.js` to quickly change:
- focus areas
- about text
- strengths
- projects
- process
- contact email

## Files

- `index.html` — page structure
- `styles.css` — visual system + responsive layout
- `data.js` — editable portfolio content
- `script.js` — dynamic rendering and simple interactions
